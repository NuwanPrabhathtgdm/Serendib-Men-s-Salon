# Serendib Salon Website

This is a premium, beautifully designed salon website with an integrated booking system built using HTML, CSS, PHP, and MySQL.

## Setup Instructions

1. **Install XAMPP** (or any local server like WAMP/MAMP).
2. **Move the project**: Copy this `serendib_salon` folder into your `C:\xampp\htdocs\` directory.
3. **Start Apache and MySQL** from the XAMPP Control Panel.
4. **Setup Database**:
   - Open your browser and go to `http://localhost/phpmyadmin`
   - Go to the **Import** tab.
   - Choose the `database.sql` file located in this project folder and click **Import**. This will create the `serendib_salon` database and the `bookings` table automatically.
5. **Run the Website**:
   - Open your browser and navigate to `http://localhost/serendib_salon`
   
## Features
- **Modern UI/UX**: Premium dark theme with gold accents, smooth animations, and responsive design.
- **Online Booking System**: Users can easily select services, dates, and times.
- **Database Integration**: All bookings are securely saved to the MySQL database via PHP.
